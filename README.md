## Banking Management System

An application for project of course Database in USTC, based on flask, bootstrap, MySQL



### Usage

***

1. Clone this repository
2. Deploy based on deployment.md
3. Read lab03_report.pdf about how to design and use this application
4. Explore and enjoy
